An example of UI application using microui (https://github.com/rxi/microui). 
This examples uses Raylib(https://www.raylib.com/) as backend.
Also uses microui-raylib bindings (https://github.com/marionauta/microui-raylib).

Requirements:
- Oberon-07 compiler with linux32o support (compilerX).
- cc compiler.
- ar archive tool.
- make.

Building:
- Use "make all".

Usage:
- Run ./bin/HelloGui or ./bin/HelloGui "window title"
